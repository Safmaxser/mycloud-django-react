# Changelog

Все заметные изменения в проекте MyCloud будут фиксироваться в этом файле.
Формат основан на [Keep a Changelog](https://keepachangelog.com).

## [1.5-mobile] — 11.03.2026
### Added
- Полная адаптивная вёрстка (Responsive UI) для смартфонов и планшетов.
- Оптимизация отображения таблиц и модальных окон под узкие экраны и различные ориентации устройств (**Portrait/Landscape**).
- Реализация адаптивной навигации (Sidebar), корректно подстраивающейся под высоту и ширину экрана мобильных устройств.
- Использование медиа-запросов Tailwind CSS для обеспечения стабильности интерфейса при повороте экрана.

## [1.4-infra] — 05.03.2026
### Added
- Автоматизированный цикл Continuous Deployment (CD) через GitHub Actions (SSH-деплой на продакшн-сервер).
- Интеграция Traefik в качестве Edge Router для автоматического выпуска и продления SSL-сертификатов (Let's Encrypt).
- Настройка защищённой маршрутизации трафика и скрытие внутренних сервисов бэкенда.
- Полный комплект технической документации: README.md (руководство по развёртыванию), CHANGELOG.md (история изменений) и MIT License.
- Оптимизация Docker Compose конфигурации для работы в продакшн-окружении.

## [1.3-qa] — 03.03.2026
### Added
- Полное тестовое покрытие (100% Coverage) для Backend (Pytest) и Frontend (Vitest).
- Интеграция CI-пайплайна в GitHub Actions для автоматической проверки тестов и линтинга.

## [1.2-realtime] — 24.02.2026
### Added
- Слой реального времени на базе Django Channels (ASGI) и Redis.
- WebSocket-события для синхронизации создания/удаления файлов и обновления квот.
- Кастомный `socketMiddleware` на фронтенде для реактивного обновления Redux Store.

## [1.1-frontend] — 21.02.2026
### Added
- Полная реализация интерфейса на React 19 и Tailwind CSS.
- Система управления состоянием через Redux Toolkit.
- UI Kit: набор переиспользуемых компонентов (модалки, таблицы, инпуты).
- Обработка ошибок API через Axios-интерцепторы.

## [1.0-backend] — 04.02.2026
### Added
- Ядро системы на Django 6.0 и Django REST Framework.
- Архитектура безопасной отдачи файлов через Nginx `X-Accel-Redirect`.
- Система логирования и многоуровневая фильтрация файлов.

## [0.1-setup] — 28.01.2026
### Added
- Базовая инфраструктура в Docker Compose (Python, Node.js, Postgres).
- Настройка сред разработки и продакшена.
